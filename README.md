# 🚀 HRMS System (MERN + Microservices + Kafka + Docker)

## 📌 Overview
A full-stack HR Management System built using MERN stack with microservices architecture, Kafka event-driven system, and Docker deployment.

---

## 🧠 Features
- 🔐 JWT Authentication (Admin / Employee)
- 👤 Employee Management
- 📅 Leave Management System
- ⚡ GraphQL + REST APIs
- 🔄 Kafka Event Streaming (Leave Notifications)
- 🐳 Dockerized Full System

---

## 🏗️ Architecture
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Messaging: Kafka
- Deployment: Docker
